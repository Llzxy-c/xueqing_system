# 智学·学情分析系统

> 🎓 数据看板 + AI 教学助手 | 课堂智能决策

一个面向基础教育场景的智能教学辅助平台，将学生考试成绩自动转化为可视化图表，并通过 AI 生成专业学情分析报告与教学改进建议，帮助教师从繁琐的数据处理中解放出来，专注于更有温度的教学工作。

---

## 📖 项目背景

传统学情分析依赖教师手动统计成绩、制作图表、撰写报告，耗时长、效率低，且教学建议容易流于空泛。本项目利用低代码工具和国产大模型，搭建了一套“数据可视化 + AI 诊断”一体化平台，让普通教师也能轻松实现数据驱动的精准教学。

---

## ✨ 核心功能

### 📊 学情数据多维可视化

基于 Metabase 开源 BI 引擎，支持上传 CSV/Excel 格式的成绩数据，自动生成多种交互式图表：

- 班级平均分对比柱状图
- 分数段人数分布饼图
- 各学科平均分横向对比图
- 学生个人成绩趋势折线图

所有图表整合至统一仪表板，支持公开链接嵌入，与 AI 助手同屏展示。

### 🤖 AI 学情诊断与策略生成

接入扣子智能体平台 + 豆包大语言模型，构建专属“学情分析助手”：

- 内置专业提示词，角色设定为 20 年教龄教研专家
- 自动生成三段式分析报告：整体综述 → 亮点与问题诊断 → 教学改进建议
- 每条建议均说明“针对什么问题”和“具体怎么做”，避免空泛建议
- 通过 Coze Web SDK 嵌入页面，即开即用，无需跳转

### 🎨 一体化前端整合

- 纯静态 HTML/CSS/JS 实现，单文件即用
- 左右分栏布局：左侧看板 + 右侧 AI 助手
- 毛玻璃 UI 设计：渐变背景、半透明面板、悬停动画
- 骨架屏加载动画，优化加载体验

### 🆓 零成本国产化方案

- **Metabase**：开源免费版，Docker 一键部署
- **扣子平台**：国内直连，无需翻墙
- **豆包大模型**：免费调用额度
- **数据本地存储**：学生成绩不上传第三方，符合教育数据隐私要求
- **一台电脑即可运行**：无需服务器，无需软件授权

## 🚀 快速开始

### 环境要求

- [Docker Desktop](https://www.docker.com/products/docker-desktop/)（运行 Metabase）
- Python 3.x（启动本地 Web 服务器，可选）
- 现代浏览器（Chrome / Edge / Firefox）

### 部署步骤

#### 第一步：启动 Metabase

```bash
docker run -d --name my_metabase -p 3000:3000 metabase/metabase
```

等待约 1 分钟后，访问 http://localhost:3000：

1. 创建管理员账号
2. 上传成绩 CSV 文件（参考模板见下方）
3. 创建可视化图表和仪表板
4. 开启「管理员设置 → 嵌入 → 静态嵌入」
5. 获取仪表板公开链接

第二步：创建扣子智能体

1. 注册 扣子平台
2. 创建 Bot，名称设为“学情分析助手”
3. 在「人设与回复逻辑」中填入以下提示词：

```
# 角色
你是一位拥有20年教学经验的资深教育数据分析专家，擅长从考试数据中洞察学情问题，并提出切实可行的教学改进建议。

# 任务
根据用户提供的班级考试成绩数据，撰写一份专业的学情分析报告。

# 报告结构
1. **整体表现综述**：描述班级平均分、最高分、最低分，评估整体水平
2. **亮点与问题诊断**：指出成绩分布特点，定位核心问题
3. **教学改进建议**：提供 3 条具体、可落地的教学措施

# 输出要求
- 使用专业但易懂的语言
- 结论需有数据支撑
- 建议具体可操作，避免空泛
- 语气积极建设性
```

4. 发布 Bot，获取 Bot ID 和个人访问令牌

第三步：配置前端文件

打开 index.html，找到以下三处并替换：

占位符 替换为
YOUR_DASHBOARD_ID Metabase 仪表板的 UUID
YOUR_BOT_ID 扣子 Bot ID
YOUR_COZE_TOKEN 扣子个人访问令牌

第四步：启动平台

```bash
# 进入项目目录
cd xueqing-system

# 启动本地 Web 服务器
python -m http.server 8080
```

浏览器访问 http://localhost:8080/index.html 即可使用。
